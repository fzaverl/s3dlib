Installation¶

Dependency Requirements¶

S3Dlib requires the installation of the following:

  • Python 3.6+
  • NumPy 1.15+
  • SciPy 1.1+
  • Matplotlib 3.0+

Package Installation¶

Once the above dependencies are installed, S3Dlib can be installed using the pip installer:

pip install s3dlib

Alternatively, the S3Dlib package may be built from the source code on GitHub located at:

https://github.com/fzaverl/s3dlib
