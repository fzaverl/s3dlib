***********************
Hello World Example
***********************

.. image:: images/hw_example.png
   :class: sphx-glr-single-img

The simplest *hello world* example is found in the tutorial :ref:`hello-default`.

This example demonstrates the basics of setting object resolution (rez), functional geometric mapping, surface color mapping,
shading, highlighting, creating a color bar, string representation of a surface object, setting axis limits,
axes view orientation, and finally, adding the surface object to the 3D viewing axis.

This example is based on the Matplotlib function used in the
`Frontpage contour <https://matplotlib.org/3.1.1/gallery/frontpage/contour.html#sphx-glr-gallery-frontpage-contour-py>`_
example.
