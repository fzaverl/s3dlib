cmap\_xtra Functions
===========================


This function requires the additional installation of the
`colorspacious <https://pypi.org/project/colorspacious/>`_
package.
The :ref:`cmap_extra source <xtra-download>` 
file can be copied directly into a file and then included in the script.
See, for example, :ref:`star`.
