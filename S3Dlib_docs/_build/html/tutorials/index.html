This page contains basic tutorials for constructing 3D surface plots using S3Dlib. Familiarity with using Matplotlib and Numpy libraries is assumed. These tutorials minimize detailed formatting of plots such as inclusion of titles, labels, etc. Instead, focus is placed on usage of the S3Dlib capabilities.

For a more detailed explanation, see the user Guides pages, Also, specific usage is provided in various Examples descriptions.
