cmap_xtra Functions¶

This function requires the additional installation of the colorspacious package. The cmap_extra source file can be copied directly into a file and then included in the script. See, for example, Multiple Geometric Maps 2.

Auxiliary functions for use with Matplotlib.

cmap_xtra.Lab_cmap_gradient(lowColor='k', highColor='w', name=None, mirrored=False)[source]¶

A linear-in-Lab-space Colormap, with option of registering the map.

Parameters:
  • locColor (RGB color, optional, default: 'black') – Color at the low end of the Colormap range.
  • highColor (RGB color, optional, default: 'white') – Color at the high end of the Colormap range.
  • name (str, optional) – The registered name to identify the colormap. If it’s None, the name will be a string of random characters and the map is not registered.
  • mirrored (bool) – If True, colormap is divided into two linear segments with the lowColor at the low and high values, the highColor in the middle.
Returns:

LinearSegmentedColormap – An instance of a colormap.
